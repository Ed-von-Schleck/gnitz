Always fix issues encountered during work, even if they are pre-existing.

**Why:** The user does not care whether a bug existed before the current changes. If it's broken, fix it. Dismissing issues as "pre-existing" is not acceptable.

**How to apply:** When encountering any failing test, build error, or broken behavior during development — fix it immediately. Do not skip it, do not report it as "pre-existing", do not suggest working around it.
